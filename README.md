# 🤖 VoxMind AI – Multilingual Voice Chatbot

VoxMind AI is an intelligent, multilingual, voice-enabled chatbot that allows users to interact using both text and speech. It generates dynamic responses using AI without relying on any predefined database, making it flexible, interactive, and human-like.

---

## 🚀 Features

- 🎤 Voice Input (Speech-to-Text)
- 🔊 Voice Output (Text-to-Speech)
- 💬 Interactive Chat Interface (ChatGPT-style UI)
- 🌐 Multilingual Support (English, Hindi, Telugu)
- 🤖 AI-Powered Dynamic Responses (No Database Required)
- 🎭 Personality Modes (Friendly, Professional, Funny)
- ⏳ Typing Animation for Real-time Feel
- 💾 Chat History using Local Storage
- 🌙 Dark Mode UI

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Flask)  
- AI Integration: OpenAI API  
- Speech Processing: Web Speech API

## ▶️ How to Run the Project

1. Clone the repository:
git clone https://github.com/kotojusaichandana/voxmind-ai
cd voxmind-ai

2. Install dependencies:
pip install -r requirements.txt

3. Set API Key (Important)

Windows (PowerShell):
$envOPENAI_API_KEY="your_api_key_here"


4. Run the application:
python app.py

5. Open in browser:
http://127.0.0.1:5000

## 🎯 Project Objective

The goal of this project is to build a real-time AI chatbot that:
- Works without a traditional database  
- Supports voice-based interaction  
- Provides multilingual communication  
- Delivers a user-friendly and interactive experience  

---

## 🌟 Unique Highlights

- Real-time AI conversation (like ChatGPT)
- Voice-enabled interaction for accessibility
- Language selection for wider usability
- Clean and modern UI design
- No hardcoded responses or datasets

---

## ⚠️ Limitations

- Voice output depends on browser-supported voices  
- Some languages (like Telugu/Hindi) may have limited voice support  
- Requires internet connection for AI responses
